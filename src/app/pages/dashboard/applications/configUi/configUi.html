Liaison Configuration Manager Information Architecture Interaction Design UI/UX

An online system developed to help organizations and institutions customize their applicant-facing CAS (Centralized Application Service) instance, called the Applicant Portal. In the Configuration Manager an administrator of a CAS can create a new application, alter the branding of their application, as well as input what kinds of documents, transcripts, and questions they would like to ask potential applicants to submit.

This product is a part of Liaison International, and works in conjunction with the Applicant Portal and Applicant Gateway.

This guide covers some things to consider when adding or updating application metadata for the Liaison UNICAS repository. It's far from complete and comments or suggestions are welcome.

  1. clone the UNICAS Repository to new work space
  2. git clone https://github.com/Liaison-Intl/unicas.git
  3. open the UNICAS Repo you just cloned
